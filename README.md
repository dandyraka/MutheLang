# MutheLang

Example:  
`Aiden depre` = `ade`  
`gaiden` = `ga`  
`maiden upru` = `mau`  
`maiden kaiden naiden nes` = `makanan`  
`saiden tepre` = `sate`  
`copro baiden ipri nes` = `cobain`  
`supro aiden piprines` = `suapin`  
`maiden maiden` = `mama`  
`taiden ngaiden nes` = `tangan`  
`peregeteles` = `pegel`  
`maiden tupru les` = `mantul`
