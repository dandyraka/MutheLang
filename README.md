# Umandanify

## Cara penggunaan
- NodeJS  
Install `npm i`  
Run `npm start`

- Python  
`python umandanify.py < input.txt` atau  
`echo "teks yang ingin dikonversi" | python umandanify.py`

------

### Rumus:  
- suku kata a + iden  
- suku kata i + pri  
- suku kata u + pru  
- suku kata e + pre  
- suku kata o + pro  
- huruf mati + es  
- ng mati = strengen (ex sayang => saiden yaiden strengen)  
- ng hidup = ngaiden (ex tangan => taiden ngaiden nes)
